Working with Functions

Named Function

A named function is a set of related statements that is assigned a name. You can use this name to reference, or call, this set of statements in other parts of your code. Generally, you use a named function when you want to be able to reuse the function statements within your code.

Anonymous Function

The statements in an anonymous function work only in a single context—the place in the code where they are located. You cannot reference an anonymous function anywhere else in your code. Generally, you use an anonymous function for statements that you need to run only once.

The return Statement

In many instances, you may want your code to receive the results from a called function and then use those results in other code. For instance, consider a function that calculates the average of a series of numbers that are passed to it as arguments. Such a function would be useless if your code could not print or use the result elsewhere. As another example, suppose that you have created a function that simply prints the name of a student. Now suppose that you want to alter the code so it uses the student name in another section of code.
