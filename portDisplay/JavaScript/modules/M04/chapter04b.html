04B: Logic Errors

Testing and Debugging

Sometimes the most difficult errors to find are logic errors. Symptoms of a logic error include incorrect output and unpredictable program behavior. The most effective techniques for debugging logic errors include a structured walkthrough, debug "write" statements, and console logging. Use these techniques to correct the logic errors in this script.
